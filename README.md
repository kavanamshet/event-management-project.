# 🎉 Event Management System

## 📌 Overview
This project is a **DBMS-based Event Management System** developed to efficiently manage event-related activities such as event creation, participant registration, and coordinator management. It uses structured database design to ensure smooth handling of event data.

---

## 🎯 Objectives
- To simplify event organization and management  
- To maintain participant and event records efficiently  
- To reduce manual work using a database-driven system  
- To provide an easy-to-use interface for users and coordinators  

---

## 🛠️ Features
- 📅 Event creation and management  
- 👥 Participant registration system  
- 🧑‍💼 Coordinator management  
- 🗂️ Structured data storage using DBMS  
- 📊 Efficient data handling and retrieval  

---

## 🏗️ System Design
The system is designed using:
- **Entity-Relationship (ER) Diagrams**
- **Relational Database Model**
- Proper relationships between entities like:
  - Events
  - Participants
  - Coordinators
